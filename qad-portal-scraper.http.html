request

(request method url scraper & [opts])

Makes a HTTP method (e.g. :get, :post) request to the specified url using the session details established by the given scraper returning a future that will yield the response when dereferenced.

url must be relative the the base URL of the scraper.

An optional map of additional request options opts may be provided.

Re-directs will not be followed and the response body will be encoded as a byte stream.
